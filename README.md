# Центр карьеры ИРКПО (Flask + SQLite)

Стартовая структура проекта, где HTML-макеты перенесены в Flask-приложение.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

База SQLite создаётся автоматически: `instance/career_center.db`.

## Роли и тестовые аккаунты

- admin: `admin@irkpo.ru` / `admin123`
- curator: `curator@irkpo.ru` / `curator123`
- student: `student@irkpo.ru` / `student123`

## Что уже есть

- вход по email/паролю;
- роли `student`, `admin`, `curator`;
- администратор создаёт студентов;
- студент редактирует профиль и ведёт портфолио;
- куратор подтверждает/отклоняет портфолио студентов своей группы;
- админ управляет вакансиями и курсами/семинарами/практиками;
- статусы `active`, `hidden`, `archive` для вакансий и программ;
- у очных программ есть поле количества мест;
- студент видит вакансии и программы;
- публичное резюме по ссылке `/resume/<token>`;
- модальное окно для отклика с контактами и копированием ссылки резюме.
